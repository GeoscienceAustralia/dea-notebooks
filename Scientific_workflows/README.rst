Scientific workflows
====================

Scientific workflows for generating published DEA products or analysis tools.
These notebooks and scripts may require access to external files or resources 
not available within this repository. Additional set-up steps may be required 
to run any code: please refer to documentation within each sub-directory for 
instructions.

--------------------

If you use any of the notebooks, code or tools in this repository in your work, please reference them using the following citation:
